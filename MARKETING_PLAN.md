# Vaulted Launch Plan

## Positioning

**Tagline:** "Your net worth tracker, not your data.
"

**Subline:** "Track assets and liabilities without linking your bank or creating an account."

## Core differentiator

- 100% local storage (no server writes)
- no bank login required
- currency support for global users
- simple mobile-first workflows

## 30-minute launch sprint

1. Prepare launch assets
2. Publish landing screenshots / short demo
3. Run launch posts in 4 communities
4. Set up waitlist and conversion metric

## Launch copy (short)

- **Website hero:** "Track your net worth without giving your finances away."
- **Social:** "Made a privacy-first net worth tracker: no sign-up, no bank integration, no server-side data."

## Post-launch KPIs (first 7 days)

- Reddit replies + feedback threads
- 20+ waitlist entries (if we add waitlist)
- 10+ people import CSV/entries in first day
- 3+ recurring feature requests we can turn into roadmap items

## Next experiments after launch

1. Add quick import templates for broker CSVs
2. Add currency conversion presets per profile
3. Add optional encrypted cloud backup (opt-in)
4. Native-style installation prompt and app icon polish
